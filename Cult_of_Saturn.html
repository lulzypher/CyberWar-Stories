Start Here — Plain English Overview

What you have on this drive is a self-contained alternate-reality game (ARG) — or LARP packaged as research — built around FalseProphet-HQ, a website that was live for years and is now offline. The /x/ thread and DISK001 dump are someone handing out the whole puzzle box.

Three layers: the wrapper (this folder), the game (Flash SWF maze), and the ideas underneath (Cult of Saturn, Worlds.com, esoteric lore).

The wrapper — what DISK001 is

  • Cult of Saturn material — Frank Webster, Worlds.com, occult threads
  • FalseProphet-HQ mirror — Flash .swf pages instead of normal HTML
  • Side archives — poems, Worlds 2008, occult PDFs (Books.rar), Frank Webster dumps

Passwords are thematic, not cryptographic. See the table in Archive Passwords.

The game — how FalseProphet-HQ works

Each URL was one .swf file. Intended flow:

  1. login.swf — countdown + oneeyedjack → issue menu
  2. login-level2.swf — masterkey → deeper tier
  3. Issues 1–18 + issueX — newsletter chapters, puzzles, weird links
  4. Rooms — reversals (15↔51, 16↔61, 17↔71), room666 hub
  5. Keys — silverkey → goldkey → highkey/lowkey → Book gates

Wrong answers often redirect to cursed URLs (Time Cube, Jodi.org, etc.) — see Redirect Pool.

What is a .swf file? (30-second version)

.swf = Shockwave Flash. Old web format (2000s–2010s). Each file on falseprophet-hq.com was one Flash animation/page — not a normal HTML website.

Think of each issue1.swf, login.swf, room666.swf as a single screen on the old site. Click buttons inside → jump to another .swf. Type passwords in text boxes → unlock the next screen.

Flash is dead in browsers now. You need a viewer (Step 5).

Next → Step 5: install something to open them.

5

Pick a viewer (choose one)

JPEXS FFDec (recommended)
Free. Opens .swf, lets you play it in a window and read the code behind it. Download: free-decompiler.com/flash
Ruffle
Flash emulator. Good for playback in browser or desktop. ruffle.rs
Flash Player Projector
Legacy standalone player (archived). Works for simple playback if you can find a safe copy.

Easiest start: install FFDec → File → Open → pick a .swf → hit the ▶ play button in the preview pane.

Next → Step 6: open the front door of the site.

Phase 5 — Investigation path (read the cheat sheet)
9

What is _swf_scripts_all? (don't panic)

If your DISK001 copy includes a folder called _swf_scripts_all, that is not part of the original game. Someone ran every .swf through FFDec and exported the hidden code to text files — a pre-made cheat sheet so you don't have to decompile 200+ files yourself.

Phase 6 — How to search without going insane

When you have a specific question, search first, read second. Don't browse folders alphabetically.

In File Explorer (whole DISK001 folder)

Windows: select the DISK001 folder → search box → type a keyword like saturncult or bigcartel. Jump straight to the file that mentions it.

Inside one SWF's script folder

Open Notepad++, VS Code, or even Notepad → Find in Files → search for:

  • myPassword — gate passwords
  • getUrl or navigateToURL — where clicks go
  • == " — text comparisons / unlock checks

Files you can safely ignore

Anything named ScrollThumb, ScrollArrow, or inside fl\controls\ — that's Adobe Flash UI junk, not game content. Ignore the entire WayBack Machine\ tree until you need a specific historical copy.
